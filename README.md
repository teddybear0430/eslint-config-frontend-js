# eslint-config-frontend-js
ESLint config for JavaScript made by Karukichi Soejima.

## Installing

Using npm:

```
npm i -D @karukichi-soejima/eslint-config-frontend-js
```

Using yarn:

```
yarn add -D @karukichi-soejima/eslint-config-frontend-js
```
